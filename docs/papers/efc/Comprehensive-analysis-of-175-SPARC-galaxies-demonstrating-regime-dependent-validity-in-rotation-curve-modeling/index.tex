\section{EFC-R Regime Model}

The Energy-Flow Cosmology Regime (EFC-R) framework extends the baseline EFC model by introducing a regime-dependent decomposition:

\begin{equation}
E_{\text{total}} = E_{\text{flow}} + E_{\text{latent}}
\end{equation}

Where:
\begin{itemize}
    \item $E_{\text{flow}}$: Manifest energy flow (measurable dynamics)
    \item $E_{\text{latent}}$: Structural complexity (non-equilibrium stress)
\end{itemize}

Regimes are characterized by the balance parameter:
\begin{equation}
\alpha(L) = \frac{E_{\text{flow}}}{E_{\text{total}}}
\end{equation}

Where $L$ is the latent proxy quantifying structural complexity:
\begin{itemize}
    \item $\alpha \to 1$ (FLOW regime): Pure energy-flow dynamics
    \item $\alpha \to 0$ (LATENT regime): Structure-dominated
    \item $\alpha \approx 0.5$ (TRANSITION): Mixed dynamics
\end{itemize}

\subsection{Physical Interpretation}

\textbf{FLOW Regime ($\alpha \to 1$):}
\begin{itemize}
    \item Low structural complexity
    \item Smooth entropy gradients
    \item Dynamic equilibrium
    \item Examples: LSB galaxies, diffuse dwarfs
\end{itemize}

\textbf{LATENT Regime ($\alpha \to 0$):}
\begin{itemize}
    \item High structural complexity
    \item Strong non-equilibrium features
    \item Energy stored in structure
    \item Examples: Barred spirals, tidally disturbed systems
\end{itemize}

\textbf{TRANSITION Zone:}
\begin{itemize}
    \item Regime boundaries
    \item Mixed dynamics
    \item Critical slowing down expected
    \item Tipping point behavior
\end{itemize}

\subsection{Key Predictions}

The EFC-R framework makes specific testable predictions:

\begin{enumerate}
    \item \textbf{Universal regime boundaries:} Same $L$ thresholds across domains
    \item \textbf{Sharp transitions:} Step-like behavior, not gradual
    \item \textbf{Measurement degradation:} Uncertainty increases near boundaries
    \item \textbf{Cross-domain isomorphism:} Same pattern in economics, biology
    \item \textbf{Tipping point dynamics:} Hysteresis and critical slowing
\end{enumerate}

\section{Data Usage Guide}

\subsection{Quick Data Access}

To load and explore the classified results:

\begin{verbatim}
import json

# Load classified results
with open('data/sparc175_classified.json') as f:
    data = json.load(f)

# Access specific galaxy
galaxy = data['results']['NGC6503']
print(f"Regime: {galaxy['regime']}")
print(f"EFC chi2/dof: {galaxy['efc']['chi2_red']:.2f}")
print(f"Latent proxy: {galaxy['latent']['L']:.3f}")
\end{verbatim}

\subsection{Reproduce Complete Analysis}
