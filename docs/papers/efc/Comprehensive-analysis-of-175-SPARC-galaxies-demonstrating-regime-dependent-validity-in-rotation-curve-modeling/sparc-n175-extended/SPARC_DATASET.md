# SPARC N=175 Dataset

## Overview
Complete SPARC (Spitzer Photometry and Accurate Rotation Curves) dataset containing:
- **175 galaxies** with high-quality rotation curves
- **3391 individual rotation curve points** across all galaxies
- Distance measurements, morphological classifications, and inclination angles

## Source
- **Paper**: Lelli, F., McGaugh, S. S., & Schombert, J. M. (2016)
- **Journal**: Astronomical Journal, 152, 157
- **VizieR**: J/AJ/152/157
- **Downloaded**: 2026-01-10

## Usage in EFC

This dataset is used for:
1. Testing EFC rotation curve predictions
2. Validating entropy-gradient model against observations  
3. Comparing with ΛCDM dark matter models
4. Statistical analysis of galaxy rotation systematics

## Data Quality
- High-resolution rotation curves
- Distance-independent measurements
- Morphological diversity (dwarfs, LSBs, spirals)
- Quality-controlled photometry from Spitzer
